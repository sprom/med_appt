Welcome to StayHealthy Images Directory
======================================

This directory contains all image assets used in the StayHealthy telemedicine platform.

Structure:
/images
  ├── doctors/           - Doctor profile photos
  ├── specialties/       - Medical specialty icons
  ├── ui/                - UI component images
  └── features/          - Feature section images

Image Formats Supported:
- PNG (preferred for web)
- JPG/JPEG (for photography)
- SVG (for icons)
- WebP (for optimization)

Usage Guidelines:
1. All images should be optimized for web
2. Use descriptive file names
3. Include alt text for accessibility
4. Maintain consistent aspect ratios
5. Compress images before uploading

Current Assets:
- Doctor profiles (PNG, 300x300px recommended)
- Medical icons (SVG or PNG)
- Feature banner images (PNG, 1200x600px recommended)
- UI component images (PNG)
